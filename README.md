UniSalas — Sistema de Reserva de Salas

Aplicação web desenvolvida com Django (Python) para gerenciamento e reserva de salas em ambiente acadêmico.

🚀 Funcionalidades:
- 🔐 Login e Logout — autenticação segura de usuários
- 🏢 Cadastro de Salas — criação, edição, listagem e exclusão de salas (CRUD completo)
- 📅 Agendamento de Reservas — reserva de salas por data e horário
- ❌ Cancelamento de Reservas — gerenciamento das próprias reservas

🛠️ Tecnologias Utilizadas:
- Python 3
- Django
- SQLite
- HTML5 + Bootstrap 5
